Optional context

context?: PrimitiveRootContext<any> | ArrayRootContext<any> | MapRootContext<any>
  • Defined in core/src/model/component.ts:10

A Context that will be made available for this Component and its children.

This must be a RootContext, created via the function createContext. Contexts like the global context, the navigation contexts or implicit contexts don't need to be declared and won't be accepted here.
