Optional flex

flex?: number
  • Defined in components/src/style/full-styles.ts:202

Sets the value for grow, shrink and basis at the same time.

Example: if you want 3 elements in the screen and wants all 3 to occupy the same space, use flex: 1 for all of them. If you want the first to occupy 2/3 of the screen while the other two shares the remaining space, use flex: 4 for the first element and flex: 1 for the other two.

See: https://yogalayout.com/docs/flex
