Optional onError

  • onError(response: MapContextNode<ErrorContext<ErrorResponse>>): Actions
    • Defined in core/src/actions/send-request.ts:88

    An action factory. This needs to return the actions to run once the request fails.

    This function receives as parameter the context for the onError event, i.e. a context containing the status, statusText and data of the response.

    example
    sendRequest<User, string>(
      // ...
      onError: (response) => alert(`An error occurred. ${response.get('data')}`)
    )
